# 测试需求：微闪贴预警客户交易信息查询（多接口）

## 场景描述
查询客户号为 962020122711000002 的微闪贴预警交易信息，预警日期为 2020-12-20，模型编号为 WSTY001。

## 接口信息
- 接口1（交易信息）：/aml/wst/custTransInfo
查询预警日期 <= 案例日期下的最新预警日期的交易
- 接口2（背书转让记录）：/aml/wst/custDrftRecord
查询预警日期 <= 案例日期下的最新预警日期的背书转让记录

## 业务约束
1. 同一个客户有预警必须有交易（1:N）
2. 同一个客户有案例必须有预警（1:N）

## 约束条件
- cust_id 固定为 962020122711000002
- model_key 固定为 WSTY001
- alert_date 固定为 2020-12-20
